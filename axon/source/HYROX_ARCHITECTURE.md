## Fundamentos Fisiológicos y Biomecánicos del Rendimiento Híbrido

El rendimiento en Hyrox (8 km carrera fraccionada + 8 estaciones funcionales) exige orquestación precisa de sistemas bioenergéticos y neuromusculares. La carrera total constituye +50% del tiempo total. Sistema cardiovascular opera 70-90% FCmax.

**Adaptaciones centrales buscadas:**
- Aumento gasto cardíaco + biogénesis mitocondrial
- Mayor oxidación lípidos a alta intensidad (preserva glucógeno)
- Incremento densidad capilar local
- Eficiencia enzimas oxidativas
- VO2max >72 ml/kg/min para podio mundial masculino

**Estaciones alta dominancia mecánica** (Sled Push/Pull, Sandbag Lunges, Wall Balls):
- Metabolismo anaeróbico glucolítico láctico
- Lactato >8.5 mmol/L
- Requiere buffering intramuscular + aclaramiento lactato (MCT1/MCT4)

**Sistema fosfágenos:**
- Activación explosiva primeros 10-15s
- Burpee Broad Jumps intermitente
- Inicio ineficiente agota PCr prematuramente

## Carrera Comprometida (Compromised Running)

**Diferenciador élite.** Sostener ciclo de marcha en prefatiga neuromuscular extrema.

**Alteraciones cinemáticas por fatiga:**
- Reducción longitud de zancada
- Incremento cadencia (compensatorio, eleva costo metabólico)
- Mayor tiempo de contacto con suelo
- Menor tiempo de vuelo
- Tendón de Aquiles pierde capacidad elástica

**NIRS revela:** oxigenación muscular local (SmO2) es el verdadero factor limitante, no FC sistémica.

## Interferencia mTOR vs AMPK

**Problema molecular:**
- Fuerza → mTORC1 → síntesis proteica
- Resistencia → AMPK + PGC-1α → biogénesis mitocondrial
- AMPK inhibe mTORC1 directamente

**Mitigación:**

1. **Separación cronológica** — mínimo 6h, óptimo 24h entre estímulos opuestos
2. **Polarización SNC (High-Low)** — consolidar estrés en días High, Zona 2 estricta en días Low

## ACWR + TSS

**TSS formula:**
```
TSS = (Duración_seg × Potencia_Normalizada × Factor_Intensidad) / (FTP × 3600) × 100
```

**Zonas ACWR (EWMA 7d/28d):**

| Rango | Estado | Intervención | Riesgo |
|---|---|---|---|
| <0.8 | Subentrenamiento | Incrementar volumen o taper | Desentrenamiento |
| 0.8-1.3 | Sweet Spot | Mantener progresión | <5% |
| 1.3-1.5 | Precaución | Consolidar, monitorear HRV | Moderado-Alto |
| >1.5 | Peligro | Reducción inmediata | Crítico |

## Readiness diario

**Parámetros innegociables pre-sesión Z4+:**

1. **HRV (RMSSD)** + RHR matutina
   - HRV suprimido >48h + RHR +3-5 bpm = dominancia simpática nociva

2. **Rendimiento neuromuscular**
   - CMJ + saltos rebote
   - Tiempo contacto óptimo <200ms
   - RSI = altura vuelo / tiempo contacto
   - Caída RSI >5% = fatiga SNC confirmada

## 3 Modelos Macrociclos

### Modelo 1: Fenotipo Fuerza (Crossfitter/Bodybuilder)

**Limitación:** baja densidad mitocondrial, economía carrera pobre.

- **Fase 1 (sem 1-8):** Base oxidativa + tolerancia impacto. 4-5 sesiones/sem, 25→40 km. 80% Zona 2. Fuerza: volumen -50%, intensidad 85-90% 1RM.
- **Fase 2 (sem 9-12):** Umbral lactato + carrera comprometida temprana. Intervalos 1km Z4. Estocadas/trineo ligero + 400m.
- **Fase 3 (sem 13-16):** ROX workouts. Deload semanas 8 y 15 (-40% volumen).

### Modelo 2: Fenotipo Aeróbico (Corredor/Triatleta)

**Limitación:** fuerza absoluta deficiente. Sled 152kg = supramáximo.

- **Fase 1 (sem 1-8):** Hipertrofia funcional + tonelaje. Carrera 20-30km Z1-Z2. Peso muerto + sentadilla frontal + trineo estático. 4-8 reps.
- **Fase 2 (sem 9-12):** Conversión bioenergética. SkiErg + Row. Trineos + intervalos anaeróbicos.
- **Fase 3 (sem 13-16):** Anti-spiking + pacing. Micro-descansos cada 10m en sled.

### Modelo 3: Élite (Elite 15 / World Championship)

**Perfil:** +15h/sem, 5K <16min, DL >160kg.

- **Fase 1 (sem 1-4):** Microciclos 14 días. 60-70 km + NRR (Neuromuscular Recruitment Runs) = 25% volumen.
- **Fase 2 (sem 5-10):** Hiper-adaptación láctica. 1000m a 3:15-3:25/km + trineo supramáximo + 100 wall balls.
- **Fase 3 (sem 11-12):** Tapering parasimpático. Sobrecompensación glucógeno + PCr.

## 6 Atributos Fisiológicos (0-99)

Sistema tipo PES para clasificación atleta Axon.

| Atributo | Definición | Medición |
|---|---|---|
| **PAC** | Motor aeróbico + capacidad ventilatoria | VO2max (Cosmed K5) + 5K/10K |
| **STR** | Fuerza absoluta + tonelaje | 3RM sentadilla trasera + peso muerto |
| **POW** | Fuerza reactiva + potencia elástica | RSI + salto vertical + SkiErg parciales |
| **TRN** | Economía transición bajo carrera comprometida | Pérdida seg/km entre carrera 1 y 8 |
| **STA** | Tolerancia láctica + resistencia periférica | Sostenimiento estocadas + wall balls |
| **RES** | Resiliencia sistémica + readaptación | HRV/RMSSD + recuperación + ACWR histórico |
