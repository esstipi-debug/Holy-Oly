# Substitution Chains: Complete Mapping

**48 total substitution chains** for intelligent session adaptation based on athlete readiness.

**Degradation Levels:**
- **Level 1 (Yellow Zone):** Mild - reduce intensity slightly, technique still high
- **Level 2 (Orange Zone):** Moderate - significantly simpler movement, noticeable intensity reduction
- **Level 3 (Red Zone):** Heavy - drastically simplified or movement pattern eliminated

## SNATCH FAMILY SUBSTITUTIONS (9 chains)

### Full Snatch (Complexity 10, CNS 10)
1. **Snatch → Power Snatch**
   - Level: 1 (Yellow)
   - Reason: poor_coordination
   - Intensity Mod: -5% | Volume Mod: 0%
   - Logic: Removes receiving depth, athlete still catches bar on shoulders

2. **Snatch → Hang Power Snatch**
   - Level: 2 (Orange)
   - Reason: poor_coordination

## Implementation Guide

### For Backend Developers

1. **Seeding Substitutions:**
   ```javascript
   // Uses ExerciseSubstitution table
   // Each row: (original_id, substitute_id, degradation_level, reason, intensity_mod, volume_mod)
   // Auto-seeded via seed-adaptation.js
   ```

2. **Querying Substitutions:**
   ```javascript
   // Get all substitutions for a given exercise at a specific risk level
   const subs = await prisma.exerciseSubstitution.findMany({
     where: {
       originalExerciseId: exerciseId,
       degradationLevel: { lte: riskLevel } // Find <= risk level
     },
     include: { substitute: true },
     orderBy: { degradationLevel: 'asc' }
   });
   ```

3. **Selecting Best Substitute:**
   - Filter by degradation level matching risk zone
   - Consider primary risk factor (coordination, fatigue, soreness)
   - Prefer substitutions matching the reason
   - Fallback to "general" reason if no match

### For Coaches

- **Yellow Zone:** Mild degradations maintain 95%+ of training stimulus
- **Orange Zone:** Moderate degradations reduce complexity but keep intensity
- **Red Zone:** Heavy degradations are safer alternatives, drastically different patterns

### For Athletes

- Substitutions are **automatic suggestions** based on your readiness
- Your coach reviews and approves (or modifies)
- You still get effective training, just adapted to your state
- No ego hits — it's science-based, not judgment
