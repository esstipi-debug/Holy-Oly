# VOLTA CYCLE BRAIN — CrossFit Periodización
> Macrociclo anual alineado al calendario CrossFit Chile 2026.
> Fuente: CompTrain + Calendario FaceWOD + CrossFyApp

---

## 1. Temporada anual Chile 2026

```
OCT–ENE → BASE (16 sem)
FEB     → OPEN PREP / PEAK 1 (4 sem)
MAR     → RECOVERY + LIGA PREP (3–4 sem)
ABR–MAY → STRENGTH BLOCK (6–8 sem)
JUN     → PEAK 2 / WODSTOCK PREP (3–4 sem)
JUL     → WODSTOCK WINTER (competición)
AGO–SEP → OFF-SEASON (8 sem)
```

**Regla crítica:** mínimo 3 semanas entre picos. Si hay menos de 3 semanas entre eventos, el engine alerta al coach y reprograma.

## 3. Bloques por fase

### BLOQUE 1 — BASE (Oct–Ene, 16 semanas)
**Objetivo:** Construir motor aeróbico, consistencia, base de fuerza.

| Variable | Target |
|----------|--------|
| Dominio principal | Engine (aeróbico) + Gymnastics |
| Intensidad metcon | RPE 6–7 (zona naranja CompTrain) |
| Volumen | Alto. 5 sesiones/semana |
| Wise Score driver | Consistency + Engine Score |
| Déficit calórico | No. Superávit leve si hay objetivo de masa |

**Qué NO hacer:** Singles pesados frecuentes. Benchmark testing. Metcons >RPE 8.

---

### BLOQUE 2 — OPEN PREP / PEAK 1 (Feb, 4 semanas)
**Objetivo:** Expresar fitness acumulado. Benchmark percentile máximo.

| Variable | Target |
|----------|--------|
| Dominio principal | Benchmark + Power |
| Intensidad metcon | RPE 8–9 en piezas clave |
| Volumen | Medio–bajo. 4 sesiones/semana |
| Wise Score driver | Benchmark Percentile |
| ACWR target | 0.8–1.0 semana Open |

**Regla Open:** 3 semanas pre-Open sin competiciones ni tests pesados.
**Post-Open:** 1 semana deload obligatoria antes de Liga prep.

---

### BLOQUE 3 — RECOVERY + LIGA PREP (Mar, 3–4 semanas)
**Objetivo:** Restaurar fatiga del Open. Preparar base para liga larga.

| Variable | Target |
|----------|--------|
| Intensidad | RPE 5–6 |
| Volumen | Bajo–medio |
| Énfasis | Movilidad + técnica + aeróbico suave |
| Wise Score | Mantenimiento |

---

### BLOQUE 4 — STRENGTH BLOCK (Abr–May, 6–8 semanas)
**Objetivo:** Subir Strength Index. Mantener Engine. Liga como test.

| Variable | Target |
|----------|--------|
| Dominio principal | Strength + Power |
| Intensidad fuerza | 80–90% 1RM en movimientos clave |
| Metcon | RPE 7–8, time domains medios (8–15 min) |
| Volumen | Medio. 5 sesiones/semana |
| Wise Score driver | Strength Index |
| Eventos liga | Usar como benchmark real, no como pico máximo |

---

### BLOQUE 5 — PEAK 2 / WODSTOCK PREP (Jun, 3–4 semanas)
**Objetivo:** Segundo pico anual. Wodstock o final de liga.

| Variable | Target |
|----------|--------|
| Estructura | Igual que Open Prep pero con más base de fuerza |
| Volumen | Bajo. 3–4 sesiones/semana |
| Intensidad | RPE 8–9 en piezas específicas |
| ACWR target | 0.8–1.0 semana competición |
| Wise Score driver | Benchmark + Strength combinados |

---

### BLOQUE 6 — OFF-SEASON (Ago–Sep, 8 semanas)
**Objetivo:** Recuperación real. Rebuild sin presión de rendimiento.

| Variable | Target |
|----------|--------|
| Intensidad | RPE 5–6 máximo |
| Volumen | 3–4 sesiones/semana |
| Énfasis | Técnica, movilidad, trabajo en debilidades |
| Wise Score | No se mide activamente |

## 4. Niveles de atleta en Volta

| Nivel | Descripción | Volumen base | Metcon target |
|-------|-------------|-------------|---------------|
| **1 — Scaled** | Aprendiendo movimientos, < 1 año CF | 3 ses/sem | RPE 6, scaled siempre |
| **2 — Rx** | Completa WODs Rx, no compite aún | 4 ses/sem | RPE 7, Rx frecuente |
| **3 — Competidor local** | Gaman League, ligas regionales | 5 ses/sem | RPE 7–8, test benchmarks |
| **4 — Competidor nacional** | Open clasificado, Wodstock | 5–6 ses/sem | RPE 8–9, picos controlados |

**El bloque y la carga se ajustan automáticamente según el nivel del atleta.**

---

## 5. Dominios CompTrain (7)

Cada sesión se clasifica en uno o más dominios. El V-Domain Engine trackea balance semanal.

| # | Dominio | Descripción |
|---|---------|-------------|
| 1 | Cardio Endurance | Piezas largas >15 min, RPE 6–7 |
| 2 | Stamina | Intervalos medios 8–15 min, RPE 7–8 |
| 3 | Strength | Trabajo de fuerza máxima, 1–5 reps |
| 4 | Power | Movimientos explosivos, tiempo corto |
| 5 | Speed & Agility | Sprints, cambios de dirección |
| 6 | Coordination / Gymnastics | Habilidades técnicas: MU, HSPU, pistols |
| 7 | Flexibility / Mobility | Movilidad activa, recuperación |

**Balance saludable por bloque:**
- Base: dominios 1, 2, 6 predominantes
- Strength Block: dominio 3, 4 predominantes
- Open/Peak: dominios 2, 4, 6 predominantes

## 6. Integración con otros engines

| Engine | Interacción |
|--------|-------------|
| **Wise Score** | Sube progresivamente por bloque. Cada bloque tiene un driver principal |
| **V-Stress Engine** | Banister adaptado: session load incluye componente cardio + muscular |
| **Competition Engine** | ACWR + readiness en semanas de Open / Wodstock |
| **Nutrition Engine** | Superávit en Base, mantenimiento en Peak, carbos altos días metcon intenso |
| **Readiness Engine** | Gate pre-sesión en todos los bloques. Más crítico en Peak |

---

## 7. Alerta de solapamiento

Si hay < 3 semanas entre eventos registrados → alerta automática al coach:
> "Solo {days} días entre {event1} y {event2}. Riesgo de sobreentrenamiento. Recomienda espaciar o priorizar uno."
