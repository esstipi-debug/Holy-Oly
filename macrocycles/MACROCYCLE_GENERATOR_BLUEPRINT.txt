SECCION 0: MAPA DE OPORTUNIDADES ML vs REGLAS
================================================================================

Cada componente del generador se marca con:

  [RULES]  = Logica deterministica, codificable con reglas fijas
  [ML]     = Machine Learning mejora significativamente el resultado
  [RAG]    = Retrieval-Augmented Generation (buscar en base de conocimiento)
  [HYBRID] = Reglas base + ML para optimizar parametros

================================================================================
SECCION 1: DNA DE ESCUELAS — REGLAS EXTRAIDAS
================================================================================

Cada escuela tiene un "DNA" con 8 dimensiones:
  1. Filosofia central (principio rector)
  2. Estructura temporal (mesociclos, semanas, dias)
  3. Patron de intensidad (como sube el %)
  4. Patron de volumen (como cambia el tonnage)
  5. Ejercicios permitidos y rotacion
  6. Formato de sesion (slots, tipos de dia)
  7. Reglas de progresion y descarga
  8. Nivel minimo del atleta

────────────────────────────────────────────────────────────────────────────────
1.1 ESCUELA CUBANA
────────────────────────────────────────────────────────────────────────────────

FILOSOFIA: "Desarrollo progresivo por niveles, jerarquia de slots"
FUENTE: Sistema cubano de halterofilia (5 planes por nivel)
VARIANTES: Novicio 2d, Novicio 3d, Intermedio 5d, Avanzado 5d, Competidor 5d+dobles

ESTRUCTURA TEMPORAL:
  - Ciclo: 16 semanas (4 meses)
  - Mesociclo: 4 semanas cada uno (4 meses)
  - Patron semanal: 3x1 (3 semanas carga + 1 descarga)
  - Fases: Cimentacion → Desarrollo → Intensificacion → Realizacion

PATRON DE INTENSIDAD (IMR = Intensidad Media Relativa):
  - Mes 1 (Cimentacion):  IMR 60-62% (novicio) / 70-72% (avanzado)
  - Mes 2 (Desarrollo):   IMR 64-66% / 74-76%
  - Mes 3 (Intensificacion): IMR 68-70% / 78-82%
  - Mes 4 (Realizacion):  IMR 72%+ / 85-95%+

  Dentro de cada mes (patron 3x1):
    Sem 1 (Base):    IMR base
    Sem 2 (Carga):   IMR + 2%
    Sem 3 (Impacto):  IMR + 5%
    Sem 4 (Descarga): IMR - 10% (caida dramatica)

PATRON DE VOLUMEN:
  - Sem 1: 100% volumen base
  - Sem 2: 115-120% (sube sets o reps)
  - Sem 3: 130-140% (pico de volumen)
  - Sem 4: 50-60% (descarga, -50%)

  Progresion entre meses: volumen BAJA mientras intensidad SUBE
    Mes 1: ~35 reps/sesion
    Mes 2: ~28 reps/sesion
    Mes 3: ~20 reps/sesion
    Mes 4: ~12 reps/sesion

JERARQUIA DE SLOTS (CLAVE CUBANA):              [RULES]
  Slot 1: ARRANQUE (Snatch y variantes)
  Slot 2: ENVION (Clean & Jerk y variantes)
  Slot 3: FUERZA (Squat, Pull, Press)

  Regla: Siempre en ese orden. Nunca fuerza antes de tecnico.

TIPOS DE DIA (para 5 dias/semana):              [RULES]
  - LUN: VOL (35% carga semanal) — Volumen alto
  - MAR: AUX (22%) — Auxiliares y variantes
  - MIE: MED (30%) — Intensidad media
  - JUE: TEC (15%) — Tecnico/recuperacion activa
  - VIE: INT (28%) — Intensidad alta / test

EJERCICIOS POR NIVEL:                            [RULES]
  Novicio:
    Slot 1: hang_power_snatch, power_snatch, muscle_snatch
    Slot 2: hang_power_clean, power_clean_jerk, push_press
    Slot 3: back_squat, front_squat, snatch_deadlift, overhead_squat

  Intermedio (agrega):
    Slot 1: + snatch, no_foot_snatch, block_snatch, snatch_balance
    Slot 2: + clean_jerk, push_jerk, block_clean, jerk_dip
    Slot 3: + clean_deadlift, snatch_high_pull, good_mornings

  Avanzado (agrega):
    Slot 1: + halting_snatch_dl, deficit_snatch_pull
    Slot 2: + jerk_drive, jerk_recovery, behind_neck_jerk
    Slot 3: + hip_thrust, box_jumps, pliometria

  Competidor (agrega):
    Doble turno: AM solo arranque, PM solo envion
    Regla: UN SOLO GESTO TECNICO POR SESION

REGLAS DE PROGRESION:                            [RULES]
  - Intra-semana: sets suben (sem1: 4x3, sem2: 5x3, sem3: 6x3)
  - Inter-mes: reps bajan (mes1: x3, mes2: x2, mes3: x1)
  - Intensidad: +2-3% por semana dentro del mes
  - Descarga: -50% volumen, -10% intensidad cada 4ta semana
  - Si falla: No especificado (coach decide)

NIVEL MINIMO: novice (tiene plan de 2 dias)

--- ML OPPORTUNITY ---                           [ML]
  DONDE: Seleccionar cual nivel cubano asignar al atleta
  INPUT: meses_entrenando, compliance_rate, pr_progression_rate
  OUTPUT: nivel recomendado (novicio_2d, novicio_3d, int_5d, avanzado, competidor)
  MODELO: Decision Tree / Random Forest
  DATOS NECESARIOS: 50+ atletas con historial de progresion

  DONDE: Ajustar el patron 3x1 a la respuesta individual
  INPUT: readiness_post_semana_carga, rpe_divergence, sleep_avg
  OUTPUT: descarga en semana 3 o semana 4? duracion descarga?
  MODELO: Logistic Regression
  DATOS NECESARIOS: 100+ ciclos completados

--- RAG OPPORTUNITY ---                          [RAG]
  DONDE: Explicar al coach por que se eligio este nivel
  QUERY: "Este atleta lleva 8 meses, compliance 90%, PR +15kg en 6 meses"
  RETRIEVE: Documentacion cubana sobre criterios de nivel
  GENERATE: "Recomendamos Avanzado 5d porque..."

1.2 ESCUELA RUSA
────────────────────────────────────────────────────────────────────────────────

FILOSOFIA: "La variabilidad aguda de carga es mas efectiva que progresion lineal"
FUENTE: Lev Matveyev + Arkady Vorobyev
VARIANTES: 5 dias (intermedio), 6 dias (avanzado/competidor)

ESTRUCTURA TEMPORAL:
  - Ciclo: 16 semanas (4 mesociclos de 4 semanas)
  - Mesociclos: Hipertrofia → Fuerza Basica → Fuerza/Potencia → Peaking
  - Patron semanal: Onda rusa de 4 semanas

PATRON DE INTENSIDAD:                           [RULES]
  Mesociclo 1 (Hipertrofia):   IMR 65-70%
  Mesociclo 2 (Fuerza Basica): IMR 75-80%
  Mesociclo 3 (Fuerza/Potencia): IMR 85-90%
  Mesociclo 4 (Peaking):       IMR 92-100%+

  Variante 6 dias (avanzado):
  Mesociclo 1: IMR 68-72%
  Mesociclo 2: IMR 78-82%
  Mesociclo 3: IMR 88-92%
  Mesociclo 4: IMR 95-102%

ONDA RUSA DE VOLUMEN (LA CLAVE):                [RULES]
  Semana 1: 100% tonnage (DOBLE — maximo volumen)
  Semana 2: 90% tonnage (mantenimiento)
  Semana 3: 60% tonnage ("cae como roca")
  Semana 4: 45% tonnage (deload profundo)

  Variante 6 dias:
  Semana 1: 100% + doble frecuencia
  Semana 2: 95% + intensidad sostenida
  Semana 3: 65% + tecnica refinada
  Semana 4: 40% + recuperacion profunda

ROTACION DE GRUPOS (5 dias):                     [RULES]
  Dia 1: Grupo A — Snatch (principal, auxiliar, estructural)
  Dia 2: Grupo B — Clean (principal, auxiliar, estructural)
  Dia 3: Grupo C — Pulls (snatch pull, clean pull, deadlift)
  Dia 4: Grupo D — Squats (back, front, pause, box)
  Dia 5: Grupo E — Overhead (jerk, push press, military)

  Variante 6 dias (agrega):
  Dia 6: Grupo F — Accesorios (muscle snatch/clean, holds, mobility)
  + 2 dobles opcionales (AM/PM)

EJERCICIOS POR GRUPO + FASE:                    [RULES]
  Grupo A (Snatch):
    Hipertrofia:   hang_snatch (principal), snatch_pull (aux), overhead_squat (est)
    Fuerza:        snatch (principal), snatch_pull @ 95% (aux), snatch_balance (est)
    Potencia:      snatch (principal), deficit_snatch_pull (aux), drop_snatch (est)
    Peaking:       snatch singles (principal), speed_pull (aux), competition_sim (est)

  Grupo B (Clean):
    Hipertrofia:   power_clean (principal), clean_pull (aux), front_squat (est)
    Fuerza:        clean_jerk (principal), clean_pull @ 95% (aux), front_squat (est)
    Potencia:      clean_jerk (principal), deficit_clean_pull (aux), jerk_balance (est)
    Peaking:       clean_jerk singles, speed_pull, competition_sim

  Grupo C (Pulls):
    Hipertrofia:   snatch_deadlift 6x5, high_pull 4x5, romanian_dl 4x6
    Fuerza:        deficit_snatch_pull 6x3, clean_deadlift 4x3, bent_rows 4x4
    Potencia:      speed_pulls 5x2 @ 100%+, shrugs, power_work
    Peaking:       minimal pulls, technique focus

  Grupo D (Squats):
    Hipertrofia:   back_squat 6x6, bulgarian_split, walking_lunges
    Fuerza:        back_squat 6x4, pause_front_squat, bulgarian_split
    Potencia:      back_squat 5x2, front_squat 4x2, box_squats
    Peaking:       back_squat 3x1, front_squat 2x1

  Grupo E (Overhead):
    Hipertrofia:   push_press 6x5, jerk_from_rack 4x4, military_press 4x6
    Fuerza:        jerk_from_rack 6x2, push_press 5x3, military_press 4x3
    Potencia:      split_jerk 5x1, push_press 4x2, behind_neck_jerk
    Peaking:       jerk singles, light technique

PRESCRIPCION (sets x reps por mesociclo + tonnage): [RULES]

  tonnage_factor = onda_rusa[semana % 4]  // [1.0, 0.9, 0.6, 0.45]

  Hipertrofia:
    principal: { sets: round(6 * tonnage), reps: 5 }
    auxiliar:  { sets: round(4 * tonnage), reps: 5 }
    estructural: { sets: round(4 * tonnage), reps: 6 }

  Fuerza:
    principal: { sets: round(6 * tonnage), reps: 3 }
    auxiliar:  { sets: round(5 * tonnage), reps: 3 }
    estructural: { sets: round(4 * tonnage), reps: 4 }

  Potencia:
    principal: { sets: round(5 * tonnage), reps: 2 }
    auxiliar:  { sets: round(4 * tonnage), reps: 2 }
    estructural: { sets: round(3 * tonnage), reps: 3 }

  Peaking:
    principal: { sets: round(4 * tonnage), reps: 1 }
    auxiliar:  { sets: round(3 * tonnage), reps: 2 }
    estructural: { sets: round(2 * tonnage), reps: 2 }

--- ML OPPORTUNITY ---                           [ML]
  DONDE: Optimizar la onda rusa por atleta
  INPUT: readiness_diaria[], compliance[], rpe_reported[], sleep[]
  OUTPUT: tonnage_wave optimizada (ej: [1.0, 0.85, 0.55, 0.40] para atleta X)
  MODELO: Bayesian Optimization / Gaussian Process
  DATOS: 200+ semanas de datos de multiples atletas
  IMPACTO: Alto — la onda rusa es el corazon del sistema, optimizarla = 10-20% mejor resultado

  DONDE: Predecir cuando saltar de mesociclo
  INPUT: rpe_trend, pr_velocity, fatigue_accumulated
  OUTPUT: "listo para fuerza" / "necesita 1 semana mas de hipertrofia"
  MODELO: Time Series Classification (LSTM o XGBoost)
  DATOS: 100+ transiciones de mesociclo documentadas

--- RAG OPPORTUNITY ---                          [RAG]
  DONDE: "Por que mi atleta esta en semana de 60% tonnage?"
  RETRIEVE: Documentacion Vorobyev sobre supercompensacion
  GENERATE: "La semana 3 reduce volumen para permitir supercompensacion.
             Tu atleta acumulo X tonnage en semanas 1-2, necesita esta descarga
             para que el cuerpo se adapte antes de la siguiente onda."

1.3 ESCUELA COLOMBIANA
────────────────────────────────────────────────────────────────────────────────

FILOSOFIA: "Piernas Primero, Intensidad Despues" — Metodo Maria Urrutia
FUENTE: Escuela colombiana (Maria Urrutia, medalla oro Sydney 2000)
VARIANTES: 4d, 5d, 6d

ESTRUCTURA TEMPORAL:
  - Ciclo: 24 semanas (6 meses) — el mas largo
  - 4 Fases:
    Preparatorio General (8 sem): Volumen extremo piernas
    Preparatorio Especial (8 sem): Balance tecnica-fuerza
    Precompetencia (4 sem): Intensidad maxima, reduccion volumen
    Realizacion (4 sem): Peak absoluto

PATRON DE INTENSIDAD:                           [RULES]
  Fase 1 (Prep General):  IMR 58-64%
  Fase 2 (Prep Especial): IMR 65-70%
  Fase 3 (Precompetencia): IMR 72-80%
  Fase 4 (Realizacion):   IMR 82-100%+

PATRON DE VOLUMEN (EXTREMO):                    [RULES]
  LA REGLA UNICA COLOMBIANA: Volumen de cuclillas

  Fase 1: 400-527 reps cuclillas/semana (RECORD METODOLOGICO)
  Fase 2: 250-350 reps cuclillas/semana
  Fase 3: 50-150 reps cuclillas/semana
  Fase 4: 0-10 reps cuclillas/semana (ELIMINACION TOTAL)

  Total ciclo: ~6,500 reps cuclillas en preparacion → 0 en competencia

  Patron semanal (dentro de cada fase):
    Semana 1-3: Carga progresiva (sube 10-15% por semana)
    Semana 4: Descarga (-40-50%)

EJERCICIOS POR FASE:                             [RULES]
  Fase 1 (Prep General):
    Tecnico: hang_power_snatch, hang_power_clean, power_jerk (BAJO %)
    Piernas CORE: back_squat 8x6, front_squat 7x6 (VOLUMEN BRUTAL)
    Piernas SEC: walking_lunges, bulgarian_split, step_ups, lateral_lunges

  Fase 2 (Prep Especial):
    Tecnico: power_snatch, power_clean_jerk (sube intensidad)
    Piernas: back_squat 5x4, front_squat 5x3 (baja volumen)
    Jerk: split_jerk, push_jerk (aparece con fuerza)

  Fase 3 (Precompetencia):
    Tecnico: snatch, clean_jerk (movimientos completos)
    Piernas: back_squat 3x2, front_squat 2x2 (minimo)
    Test: singles competitivos

  Fase 4 (Realizacion):
    Solo: snatch, clean_jerk (singles @ 95%+)
    Piernas: CERO (la fuerza YA esta construida)
    Focus: mental, recovery, peaking

REGLA CLAVE: Correlacion directa fuerza piernas = exito jerk [RULES]
  back_squat_volume * 0.7 = estimacion de fuerza de jerk
  Si BS < 1.5x CJ, hay deficit de piernas → mas volumen

--- ML OPPORTUNITY ---                           [ML]
  DONDE: Determinar cuanto volumen de cuclillas necesita este atleta
  INPUT: current_bs_1rm, current_cj_1rm, bs_cj_ratio, body_weight, age
  OUTPUT: squat_reps_per_week_target (ej: 420 para este atleta, 380 para otro)
  MODELO: Linear Regression / Random Forest
  DATOS: ratio BS/CJ vs resultado competitivo de 50+ atletas
  IMPACTO: Muy alto — sobreentrenar piernas = lesion, subentrenar = no hay jerk

  DONDE: Predecir cuando la reduccion de volumen debe empezar
  INPUT: fatigue_trend, squat_1rm_trend, jerk_performance
  OUTPUT: "empezar reduccion en semana X" (en vez del fijo semana 9)
  MODELO: Time Series Regression
  DATOS: Curvas de fatiga/performance de atletas que completaron ciclo colombiano

1.4 ESCUELA POLACA
────────────────────────────────────────────────────────────────────────────────

FILOSOFIA: "Tecnica antes peso, progresion cauta" — Zygmunt Smalcerz (1972 Olympic Champion)
FUENTE: Escuela polaca de halterofilia
VARIANTES: 4d, 5d

ESTRUCTURA TEMPORAL:
  - Ciclo: 6 semanas (el mas CORTO — ideal para principiantes)
  - 3 Fases de 2 semanas:
    Fase 1 (Reps Altas):  semanas 1-2
    Fase 2 (Reps Medias): semanas 3-4
    Fase 3 (Singles):     semanas 5-6

PATRON DE INTENSIDAD:                           [RULES]
  Fase 1: IMR 70-75%
  Fase 2: IMR 80-85%
  Fase 3: IMR 90-95%

  REGLA UNICA POLACA: +2.5kg cada semana (INVARIABLE, AUTOMATICO)
  Semana 1: 70%, Semana 2: 72.5%, Semana 3: 80%, etc.

PATRON DE VOLUMEN:                              [RULES]
  Fase 1: 66-72 reps/sesion (reps altas: 4x6, 3x8)
  Fase 2: 36-47 reps/sesion (reps medias: 4x4, 4x3)
  Fase 3: 8-16 reps/sesion (singles: 5x1, 3x1)

  Volumen cae linealmente mientras intensidad sube

EJERCICIOS:                                      [RULES]
  Dia 1: snatch + clean_jerk + front_squat
  Dia 2: power_snatch + power_clean + back_squat
  Dia 3: hang_snatch + jerk_from_rack + overhead_squat
  Dia 4: snatch_balance + clean_pull + push_press

  Siempre los mismos ejercicios, solo cambian sets/reps/%.
  Variacion MINIMA — la simplicidad ES la filosofia.

REGLAS DE PROGRESION:                            [RULES]
  1. +2.5kg por semana (automatico)
  2. BACKOFF RULE: Si fallas una serie → reduce al 80% la siguiente sesion
  3. Reps bajan cada 2 semanas: 6→4→1
  4. NO hay descarga formal (ciclo de 6 semanas es corto)

NIVEL MINIMO: novice (el mas accesible)
RIESGO DE LESION: bajo (progresion conservadora)

--- ML OPPORTUNITY ---                           [ML]
  DONDE: Ajustar el +2.5kg por semana al atleta
  INPUT: rpe_reported[], bar_speed_if_available, pr_history
  OUTPUT: incremento_semanal optimo (tal vez +1.5kg para mujer de 55kg, +3kg para hombre de 105kg)
  MODELO: Linear Regression simple
  DATOS: 30+ ciclos polacos completados
  IMPACTO: Medio — evita estancamiento o progresion demasiado agresiva

  DONDE: Predecir si necesita backoff antes de que falle
  INPUT: rpe_last_3_sessions, sleep_avg, readiness_trend
  OUTPUT: probabilidad_de_fallo en proxima sesion
  MODELO: Logistic Regression
  DATOS: 100+ sesiones con datos de RPE + exito/fallo

1.5 ESCUELA BULGARA
────────────────────────────────────────────────────────────────────────────────

FILOSOFIA: "Intensidad maxima diaria es principio absoluto" — Ivan Abadjiev
FUENTE: Daily Maxing System bulgaro
VARIANTES: 6 dias x 2-4 sesiones diarias

ESTRUCTURA TEMPORAL:
  - Ciclo: 4 semanas (ultra-corto, se repite)
  - Semanas:
    Sem 1: Establishment (establecer daily maxing)
    Sem 2: Intensification (buscar PRs)
    Sem 3: Peak Maxing (PR hunting)
    Sem 4: Deload (recuperacion)

PATRON DE INTENSIDAD:                           [RULES]
  SIEMPRE >= 95% del 1RM
  Sem 1: 95%+
  Sem 2: 95-100%
  Sem 3: 95-110% (hunting PRs)
  Sem 4: 70% (unico momento que baja)

  FLAT HIGH — no hay progresion gradual, siempre arriba

PATRON DE VOLUMEN:                              [RULES]
  ~85 reps @ 95%+ por semana (constante)
  3 sesiones por dia:
    S1 (08:00-08:30): Movimiento olimpico max
    S2 (09:00-09:30): Squat max
    S3 (17:00-17:45): Pull heavy o tecnica

  Sesion = 20-30 min MAX (no mas)
  Reps por sesion: 3-5 intentos maximos

EJERCICIOS:                                      [RULES]
  POOL MINIMO (solo 4):
    snatch, clean_jerk, front_squat, back_squat

  Variantes PROHIBIDAS: nada de muscle snatch, push press, accesorios
  Pulls solo en S3 vespertina: snatch_pull, clean_pull @ 100-120%

  Rotacion diaria:
    LUN: snatch max → front_squat max → snatch pull
    MAR: clean_jerk max → back_squat max → clean pull
    MIE: snatch max → front_squat max → tecnica ligera
    JUE: clean_jerk max → back_squat max → clean pull
    VIE: snatch max → front_squat max → competition sim
    SAB: clean_jerk max → back_squat max → heavy pull test

PROTOCOLO DE MAXING:                             [RULES]
  1. Build to max: subir gradualmente hasta max del dia
  2. 3-5 intentos @ 95%+ (solo estos cuentan)
  3. Si fallas 2 intentos seguidos → parar, pasar a siguiente sesion
  4. NO hay calentamiento formal largo (barra → 60% → 80% → 95% rapido)

REQUISITOS:                                      [RULES]
  Nivel: ELITE ONLY
  Recovery: masaje, sauna, ice baths OBLIGATORIOS
  Nutricion: surplus calorico, 2g/kg proteina
  Sueno: 9+ horas

  WARNING: Este sistema destruye a novatos. Solo atletas con 3+ anos.

--- ML OPPORTUNITY ---                           [ML]
  DONDE: Predecir el "max del dia" antes de que el atleta empiece
  INPUT: readiness, sleep, previous_day_max, fatigue_accumulated
  OUTPUT: predicted_daily_max (ej: "hoy tu snatch max sera ~97kg, no intentes 105kg")
  MODELO: Gradient Boosting / XGBoost
  DATOS: 500+ sesiones bulgaro con max diarios registrados
  IMPACTO: MUY ALTO — evita lesiones, optimiza intentos

  DONDE: Determinar si el atleta puede tolerar este sistema
  INPUT: training_age_months, injury_history, recovery_quality, compliance_rate
  OUTPUT: can_use_bulgarian (true/false), risk_score
  MODELO: Classification (Random Forest)
  DATOS: Historiales de atletas que probaron bulgaro (exito vs lesion/burnout)

PSEUDOCODIGO cycle-builder.js:

  function generateMacrocycle(schoolId, athlete, config) {
    const dna = loadSchoolDNA(schoolId);         // [RULES]
    const phases = dna.getPhases(config);         // [RULES]
    const sessions = [];

    for (const phase of phases) {
      for (let week = 1; week <= phase.weeks; week++) {
        const tonnage = dna.getTonnageFactor(week);  // [RULES]
        const weekSessions = generateWeek(
          dna, phase, week, tonnage, athlete       // [RULES]
        );
        sessions.push(...weekSessions);
      }
    }

    validate(sessions, dna.rules);                // [RULES]
    return { school: schoolId, sessions, phases, totalWeeks: sum(phases.weeks) };
  }

PSEUDOCODIGO session-builder.js:

  function generateSession(dna, phase, dayType, tonnageFactor, athlete) {
    // 1. Seleccionar ejercicios segun escuela + dia
    const exercises = dna.selectExercises(phase, dayType);  // [RULES]

    // 2. Calcular prescription
    const rx = exercises.map(ex => {
      const base = dna.getPrescription(phase, ex.category);  // [RULES]
      return {
        name: ex.name,
        sets: Math.round(base.sets * tonnageFactor),          // [RULES]
        reps: base.reps,
        intensityPct: dna.getIntensity(phase, ex.category),   // [RULES]
        weightKg: Math.round(athlete[ex.rmField] * intensity / 100 * 2) / 2  // round to 0.5kg
      };
    });

    return { exercises: rx, tonnageFactor, dayType, phase: phase.name };
  }
