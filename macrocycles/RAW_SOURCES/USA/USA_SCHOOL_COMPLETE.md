## 4 Fases Macrociclo USA

| Fase | IMR | RPE | Vol sem | Ratio Fuerza:Olímpico | Freq Snatch/C&J |
|---|---|---|---|---|---|
| **Acumulación** | 60-75% | 6.0-7.0 | 400-500 | 50:50 | 2-3 d/sem (hang/blocks) |
| **Desarrollo** | 70-80% | 7.0-8.0 | 300-400 | 40:60 | 3 d/sem (complexes) |
| **Fuerza** | 80-90% | 8.5-9.0 | 200-300 | 30:70 | 3 d/sem (full lifts) |
| **Realización** | 85-100%+ | 9.0-10.0 | <150 | 15:85 | 4-5 d/sem (alta freq) |

### Detalle por fase

**Acumulación (Mash énfasis):**
- Hipertrofia + GPP
- Variaciones hang/blocks (no desde suelo)
- Prohibido fallo técnico
- Sentadillas 3-4×/sem, priorizar trasera

**Desarrollo/Transmutación:**
- Puente neurológico fuerza → velocidad balística
- Complexes halterofilia (ej: Power Clean + Front Squat + Jerk)
- Pausas isométricas para posiciones recepción
- Sentadilla frontal gana prioridad

**Fuerza/Intensificación:**
- Resistencia acomodada (bandas/cadenas) — Mash
- Dobles y singles pesados
- Eliminar accesorios aislamiento
- Sentadilla 2×/sem para recovery SNC

**Realización/Peaking:**
- Eliminar hang/blocks → todo desde suelo
- Recovery squats solo mantenimiento
- 95-100%+ a 10-14 días competencia
- Caídas 60-70% días previos
- Singles diarios

## Fórmulas Algorítmicas

### 1. Estimación 1RM

**Epley (1985)** — óptima 1-5 reps
```
1RM = W × (1 + R/30)
```
- Uso: 3RM/5RM olímpicos + sentadilla + peso muerto
- Riesgo: sobrestima en +10 reps

**Brzycki (1993)** — conservadora
```
1RM = W / (1.0278 - 0.0278 × R)
Simplificado: 1RM = W × (36 / (37 - R))
```
- Uso: programación fase Intensificación (protege atleta)
- Ventaja: resiste sobreestimación

**Lombardi (1989)** — exponencial
```
1RM = W × R^0.10
```
- Uso: accesorios 8-15 reps (remos, zancadas, extensiones)
- NO usar en OLY clásicos

### 2. Volume Load

**Absoluto (problemático):**
```
VL_abs = Series × Reps × Carga
```
No discrimina intensidad. 10×10 con 20kg = 10×1 con 200kg en tonelaje, pero estímulo biológico abismalmente distinto.

**Relative Volume Load (RVL) — recomendado:**
```
RVL = Σ (Series_i × Reps_i × (Carga_i / 1RM_i))
```

Ejemplo: 1RM 150kg, prescrito 5×3 @ 120kg
- Intensidad relativa: 120/150 = 0.80
- RVL = 5 × 3 × 0.80 = **12.0 UA**

### 3. Zonas Intensidad (5 zonas)

| Zona | % 1RM | Propósito |
|---|---|---|
| Z1 | ≤60% | Recuperación + técnica base |
| Z2 | 61-70% | Velocidad + Powers |
| Z3 | 71-80% | Sweet spot fuerza + desarrollo |
| Z4 | 81-90% | Submáxima + mielinización |
| Z5 | 91-100%+ | Máxima + tirones comp |

**Modelo Piramidal (Acumulación):**
- Z1+Z2: 40-50%
- Z3: 35-40%
- Z4: 10-15%
- Z5: 0%

**Modelo Polarizado (Realización):**
- Z1: 60-70%
- Z2+Z3: 0% (extirpado)
- Z4+Z5: 20-30%

### 4. Compliance Score

**3 vectores:**

```
V_comp = MIN(1.0, reps_ejecutadas / reps_prescritas)

I_comp = 1 - MIN(1.0, |carga_promedio_real - carga_prescrita| / carga_prescrita)

E_comp = 1 - (|RPE_real - RPE_prescrito| / 10)
```

**Fórmula general:**
```
CS = (0.25 × V_comp) + (0.55 × I_comp) + (0.20 × E_comp)
```

**Pesos:**
- Volumen: 25% (flexible si técnica se degrada)
- Intensidad: 55% (dominante — desviaciones arruinan fase)
- RPE: 20% (amortiguador lifestyle/sueño)

**Ejemplo:**
Prescrito 5×2 BS @ 85% (130kg), RPE 8.5.
Atleta ejecuta 5×2 @ 115kg (75%), RPE 9.
- V = 10/10 = 1.0
- I = 1 - |115-130|/130 = 0.885
- E = 1 - |9-8.5|/10 = 0.95
- **CS = 0.25 + 0.486 + 0.19 = 0.926 (92.6%)**

Sistema valida: atleta mantuvo intención, adaptó inteligentemente a fatiga aguda sin romper macrociclo.
